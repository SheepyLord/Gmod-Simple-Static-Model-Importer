PMX Static Importer Runtime
===========================

Install this folder into:
  garrysmod/addons/pmx_static_importer_runtime

This addon expects imported content to exist at:
  garrysmod/data/pmx_static_importer/models/<model_id>/manifest.json
  garrysmod/data/pmx_static_importer/models/<model_id>/mesh.json
  garrysmod/materials/pmx_static_importer/<model_id>/*.png

Toolgun path:
  Construction -> PMX Static Importer
